# ZubaLink Premium Django Website

## Run on Windows

Open the folder containing `manage.py`, then run:

```powershell
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Open:
http://127.0.0.1:8000

## Included
- Django project
- Vercel config
- ZubaLink logo
- Industrial project photos
- Home, About, Services, Projects, Contact pages
- Clean 3-color corporate engineering design
